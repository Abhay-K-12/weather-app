# Weather App 🌤️

A simple and clean weather app built with HTML, CSS and JavaScript.

## Features
- 🔍 Search weather by city name
- 🌡️ Displays real-time temperature
- 💧 Shows humidity levels
- 💨 Shows wind speed
- 🌦️ Dynamic weather icons based on conditions

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## How to Use
1. Enter a city name in the search box
2. Click the search button
3. View real-time weather data!

## Note
Add your OpenWeatherMap API key in the script section to run locally.
